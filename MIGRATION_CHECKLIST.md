### 📋 需要手动迁移的内容

#### 1. 游戏资源（如果有）
**位置**: 旧项目的 `games/` 目录

**迁移步骤**:
```bash
# 复制游戏资源到新项目
cp -r ../旧项目/games/* ./public/games/

# 复制音频文件
cp -r ../旧项目/sounds/* ./public/sounds/

# 复制其他资源
cp -r ../旧项目/assets/* ./public/assets/
```

#### 2. 课程视频链接
如果旧项目有实际的课程视频URL，需要更新到数据库：

```sql
-- 在Prisma Studio或直接SQL中更新
UPDATE "Course" 
SET "videoUrl" = 'https://your-video-hosting.com/video1.mp4'
WHERE title = '基础数学入门';
```

#### 3. 产品图片
更新产品图片URL：

```bash
# 复制图片到public目录
mkdir -p public/images/products
cp 旧项目的图片/* public/images/products/
```

然后在Prisma Studio中更新Product表的imageUrl字段。

#### 4. 语言文件（如果使用多语言）
**旧项目**: lang.js
**新项目**: 可以使用next-intl或i18next

如果需要保留多语言功能，参考旧项目的lang.js创建翻译文件。
