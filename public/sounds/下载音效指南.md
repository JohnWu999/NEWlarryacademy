# 游戏音效下载指南

## 快速开始

您需要下载以下音频文件才能让游戏播放真实的动物音效：

### 📁 文件结构

```
sounds/
├── animals/
│   ├── cat.mp3      （猫叫）
│   ├── dog.mp3      （狗叫）
│   ├── rabbit.mp3   （兔子）
│   ├── panda.mp3    （熊猫）
│   ├── penguin.mp3  （企鹅）
│   ├── fox.mp3      （狐狸）
│   └── unicorn.mp3  （独角兽/魔法音效）
└── happy.mp3        （开心音效）
```

## 🎵 推荐免费音效网站

### 1. Freesound.org（推荐）⭐
- 网址：https://freesound.org/
- 特点：最大的免费音效库，音质高
- 使用：需要免费注册
- 搜索关键词：
  - Cat: "cat meow short"
  - Dog: "dog bark"
  - Rabbit: "rabbit squeak"
  - Panda: "bear cub"
  - Penguin: "penguin call"
  - Fox: "fox yelp"
  - Unicorn: "magic sparkle" 或 "fairy chime"
  - Happy: "kids cheer" 或 "yay sound"

### 2. Mixkit（最简单）⭐⭐⭐
- 网址：https://mixkit.co/free-sound-effects/
- 特点：无需注册，直接下载
- 许可：可商用，无需署名

### 3. Zapsplat
- 网址：https://www.zapsplat.com/
- 特点：音效分类清晰
- 使用：免费注册后下载

### 4. Pixabay Sound Effects
- 网址：https://pixabay.com/sound-effects/
- 特点：高质量，完全免费
- 无需署名

## 🔍 搜索技巧

### 英文关键词
- **猫**：cat meow, kitten sound, cat purr
- **狗**：dog bark, puppy bark, dog woof
- **兔子**：rabbit squeak, bunny sound
- **熊猫**：panda call, bear cub, baby bear
- **企鹅**：penguin call, penguin chirp
- **狐狸**：fox call, fox bark, fox yelp
- **独角兽**：magic chime, fairy sound, magical sparkle, unicorn (或用铃铛音效)
- **开心**：kids cheer, yay, hooray, success sound, happy cheer

## 📝 下载步骤示例（以 Freesound 为例）

1. 访问 https://freesound.org/
2. 注册免费账号（只需邮箱）
3. 搜索 "cat meow short"
4. 筛选：
   - Duration: < 2 秒
   - License: CC0（公共领域，最自由）
5. 试听音效，选择喜欢的
6. 点击下载（Download）
7. 重命名文件为 `cat.mp3`
8. 放入 `sounds/animals/` 文件夹

## 🎯 音频格式要求

- **格式**：MP3（首选）或 WAV
- **时长**：0.5 - 2 秒最佳
- **文件大小**：< 100KB 每个
- **采样率**：44.1kHz
- **音量**：中等音量（游戏会自动调整）

## 🚀 快速下载推荐

如果你想快速开始，可以在 Mixkit 上搜索这些：

1. **Cat**: "meow"
2. **Dog**: "bark" 
3. **Happy**: "cheer" 或 "success"
4. **Magic (for unicorn)**: "magic" 或 "chime"

对于兔子、熊猫、企鹅、狐狸，可以在 Freesound 上搜索对应英文名称。

## ⚠️ 注意事项

1. 确保音频文件命名完全一致（小写，正确的扩展名）
2. 文件必须是 `.mp3` 格式
3. 如果某个音效找不到，游戏会自动使用合成音效，不会报错
4. 下载时选择 CC0 或 CC-BY 许可的音效
5. 避免下载太长的音效（会影响游戏体验）

## 🎨 可选：高级自定义

如果你想使用自己录制的音效：
1. 使用 Audacity（免费软件）录音
2. 导出为 MP3 格式
3. 确保音量适中
4. 时长保持在 1-2 秒

## 💡 临时方案

如果暂时无法下载音效，游戏已内置合成音效作为备用，可以正常游玩。但添加真实音效会让游戏体验提升 100%！

## 🆘 需要帮助？

如果你在下载音效时遇到问题，可以：
1. 尝试不同的搜索关键词
2. 使用 Mixkit（最简单，无需注册）
3. 只下载部分音效（游戏会对缺失的使用合成音）
