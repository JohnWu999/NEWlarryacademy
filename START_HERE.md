## 📋 快速开始（3步完成）

### 第1步：设置数据库（5分钟）

**最简单的方式 - Supabase（免费）**：

1. 访问 https://supabase.com
2. 注册并创建新项目
3. 获取数据库连接字符串
4. 更新 `.env` 文件中的 `DATABASE_URL`

**详细步骤**：查看 [QUICK_FIX.md](QUICK_FIX.md)

---

### 第2步：初始化数据库

```bash
# 同步数据库结构
npm run db:push

# 填充测试数据
npm run db:seed
```

---

### 第3步：启动应用

```bash
npm run dev
```

访问：http://localhost:3000
